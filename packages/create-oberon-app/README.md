# create-oberon-app

`create-oberon-app` generates recipes. For a full list of recipes, please see
the monorepo README.

## Usage

Requires Node.js 22 or newer.

npx

```sh
npx create-oberon-app my-app
```

yarn

```sh
yarn create oberon-app my-app
```
