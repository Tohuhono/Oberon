# Introduction

<Callout type="info">
  Use [create-oberon-app](/docs/create-oberon-app) to scaffold a starter app.
</Callout>

OberonCMS is an extensible content management system for developers. It is built on top of the
excellent [Puck visual editor](https://puckeditor.com) and provides a powerful framework for
composing a CMS in your project.
