# Turso (sqlite)

A plugin providing a database adapter using [Turso](https://turso.tech/).

## Installation

```sh npm2yarn
npm install @oberoncms/plugin-turso
```

```tsx filename="oberon/config.ts"
import { defineConfig } from "@oberoncms/core"
import { plugin as databasePlugin } from "@oberoncms/plugin-turso"

import { clientConfig } from "./client.config"

export const config = defineConfig({
  client: clientConfig,
  plugins: [developmentPlugin, databasePlugin, authPlugin],
})
```

## Configuration

<Callout type="info">
  By default the Turso plugin will automatically use Turso for production builds and fall back to a
  development database.
</Callout>

### Environment Variables

To use Turso, you need to create a [Turso](https://turso.tech/) account and set your URL and API
environment variables:

```sh
TURSO_URL=<libsql://your-database-name.turso.io>
TURSO_TOKEN=<your-api-key>
```
