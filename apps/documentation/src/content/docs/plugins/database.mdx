# Database Adapters

Database plugins provide Oberon persistence. In practice that means page, image,
user, site, and key-value storage, plus the Auth.js adapter methods used by
Oberon auth.

Use the provider pages in this section for concrete integrations such as Turso
and PostgreSQL.

If you are implementing your own database plugin, see
[Custom > Adapter Hooks](/docs/plugins/custom/adapter-hooks).
