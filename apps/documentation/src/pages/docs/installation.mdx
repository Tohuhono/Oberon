# Installation

System Requirements:

- Node.js 20.x or later
- Next.js 14 or later

### Automatic installation

```sh npx2yarn copy
npx create-oberon-app my-app
```

Prompts will guide you through the process of selecting core plugins and setting
some sane defaults.

### Manual installation

<Callout type="warning">This is not documented or supported yet.</Callout>

```sh npm2yarn
npm i @oberoncms/core
```
