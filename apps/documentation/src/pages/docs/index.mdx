# Introduction

<Callout type="info">
  Checkout the [automatic
  installation](docs/installation#automatic-installation) to get started now.
</Callout>

OberonCMS is an extensible content management system for developers, designers
and content creators. It is built on top of the excellent
[Puck visual editor](https://puckeditor.com) and provides a powerful set of
components and plugins to build a CMS for your project.
